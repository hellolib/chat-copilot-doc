## 功能简介

“我的收藏”用于集中管理你收藏的提示词，支持搜索、筛选、编辑与一键写入。

![我的收藏](/ui-images/ui-我的收藏.png)

## 入口位置

1. 打开支持平台页面
2. 通过悬浮按钮或插件面板进入侧边栏
3. 选择“我的收藏”标签

## 常用操作

- 搜索提示词：在顶部搜索框输入关键词快速定位
- 分类筛选：按“编程 / 文本 / 图像 / PPT / 视频 / 通用”等分类筛选
- 收藏管理：点击星标取消收藏
- 一键使用：点击“写入”将提示词填入输入框
- 复制内容：点击“复制”将提示词复制到剪贴板
- 编辑收藏：点击“编辑”进入编辑界面
- 导出收藏：点击底部“导出我的收藏”按钮

## 编辑收藏

在编辑弹窗中可修改标题、分类、标签和提示词内容。

![编辑收藏](/ui-images/ui-我的收藏-编辑.png)
