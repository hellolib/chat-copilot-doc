## 功能简介

悬浮按钮是 Chat Copilot 在支持平台页面中的快捷入口，随时唤起侧边栏与核心功能，并支持自定义点击行为与悬停抽屉内容。

![悬浮按钮](/ui-images/ui-悬浮按钮.png)

## 使用方法

1. 打开任意支持的 AI 对话平台页面
2. 右下角出现 Chat Copilot 悬浮按钮
3. 点击按钮打开侧边栏，进入“提示词广场 / 我的收藏”等功能

## 常用操作

- 打开侧边栏：点击悬浮按钮
- 快速入口：在侧边栏中访问 [提示词广场](/docs/guides/plugin-square.html) 与 [我的收藏](/docs/guides/plugin-favorite.html)
- 关闭/收起：在侧边栏中点击关闭或收起按钮

## 相关设置

可在插件设置中管理悬浮按钮的显示与行为，路径为：`插件面板 → 设置 → 悬浮按钮`。

### 显示开关

开启“显示悬浮按钮”后，会在 AI 网站页面右侧展示 Chat Copilot 悬浮 Logo。

### 点击动作

可配置点击悬浮按钮时执行的动作或打开的页面：

- 无动作
- [插件配置](/docs/guides/plugin-option.html)
- [提示词广场](/docs/guides/plugin-square.html)
- [我的收藏](/docs/guides/plugin-favorite.html)
- 提示词优化


### 悬停抽屉功能

悬停在悬浮按钮上方时，会显示上侧抽屉。可自定义抽屉中显示的功能：

- [提示词广场](/docs/guides/plugin-square.html)
- [我的收藏](/docs/guides/plugin-favorite.html)
- [插件设置](/docs/guides/plugin-option.html)
- 官网
