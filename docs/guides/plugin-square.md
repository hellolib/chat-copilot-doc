## 功能简介

“提示词广场”汇集了精选提示词，支持搜索与分类筛选，帮助你快速找到可直接使用的高质量提示词。

![提示词广场](/ui-images/ui-提示词广场.png)

## 入口位置

1. 打开支持平台页面
2. 通过悬浮按钮或插件面板进入侧边栏
3. 选择“提示词广场”标签

## 常用操作

- 搜索提示词：在顶部搜索框输入关键词
- 分类筛选：按“编程 / 文本 / 图像 / PPT / 视频 / 通用”等分类筛选
- 快速写入：点击“写入”将提示词填入输入框
- 复制内容：点击“复制”将提示词复制到剪贴板
- 收藏提示词：点击星标将提示词加入“我的收藏”
