## 配置入口

在 [插件面板](/docs/guides/plugin-popup.html) 中点击“设置”，即可进入 Chat Copilot 配置页面。

<p align="center">
  <img src="/ui-images/ui-插件设置.png" alt="插件设置" width="90%" />
</p>

## 配置项说明

### 优化引擎
::: warning
内置优化引擎优化效果有限，建议使用自定义模型。
:::
选择内置优化引擎或添加自定义模型，支持配置模型提供商、API 端点与密钥等信息。
<p align="center">
  <img src="/ui-images/option/优化引擎.png" alt="优化引擎" width="90%" />
</p>

### 优化风格

选择提示词优化风格，不同风格会影响优化后的表达方式与结构。

<p align="center">
  <img src="/ui-images/option/优化风格.png" alt="优化风格" width="90%" />
</p>

### 自定义优化规则

添加专属规则，自动在每次提问时附加到提示词中。

<p align="center">
  <img src="/ui-images/option/自定义优化规则%20.png" alt="自定义优化规则" width="90%" />
</p>

### 悬浮按钮

配置悬浮按钮的显示开关、点击动作与悬停抽屉功能。

- 点击动作可选择：无动作 / 插件配置 / [提示词广场](/docs/guides/plugin-square.html) / [我的收藏](/docs/guides/plugin-favorite.html) / 提示词优化
- 悬停抽屉可配置显示项：[提示词广场](/docs/guides/plugin-square.html) / [我的收藏](/docs/guides/plugin-favorite.html) / 插件设置 / 官网

详细说明见 [悬浮按钮](/docs/guides/floating-button.html)。

<p align="center">
  <img src="/ui-images/option/悬浮按钮.png" alt="悬浮按钮" width="90%" />
</p>

### 提示词广场

可配置侧边栏显示方式，支持“挤压式”或“覆盖式”打开。功能说明见 [提示词广场](/docs/guides/plugin-square.html)。

### 快速访问

选择要在 [插件面板](/docs/guides/plugin-popup.html) 中显示的常用平台入口，支持全选、反选与重置。

### 关于

查看插件版本与相关链接（官网、GitHub、Chrome 商店、问题反馈）。
