# Chat Copilot 隐私权政策

**生效日期：** 2026-01-01

**最后更新日期：** 2026-02-12

## 概述

Chat Copilot（以下简称"本扩展"）致力于保护您的隐私权。本隐私权政策说明本扩展如何收集、使用和保护您的数据。请仔细阅读本政策以了解我们在使用本扩展时如何处理您的信息。



## 1. 数据存储位置

本扩展**仅在您的本地设备上存储数据**，我们不拥有、操作或控制任何外部服务器。

所有用户数据均通过 Chrome 浏览器的 `chrome.storage.local` API 存储在您设备的本地存储中。这些数据仅在同一浏览器的配置文件中可访问，不会上传到任何由我们控制的服务器。



## 2. 收集的数据类型

### 2.1 用户设置

我们存储以下设置以提供个性化体验：

| 数据类型 | 说明 | 是否必需 |
||||
| 语言偏好 | 您选择的语言设置（如中文、英文等） | 否 |
| 主题设置 | 界面主题偏好（如深色/浅色模式） | 否 |
| 快速访问站点 | 您启用的 AI 平台快捷方式列表 | 否 |
| 浮动按钮设置 | 浮动按钮的显示位置和行为偏好 | 否 |
| 侧边栏设置 | 提示词侧边栏的宽度、显示模式等 | 否 |

### 2.2 API 配置

当您选择使用自己的 AI 模型进行提示词优化时，我们存储：

| 数据类型 | 说明 | 是否必需 |
||||
| API 端点 | 您选择的 AI 服务提供商的 API 地址 | 是 |
| API 密钥 | 用于访问相应 AI 服务的密钥 | 是(通常) |
| 模型名称 | 使用的具体模型标识 | 是 |
| 其他参数 | 如最大 token 数、温度等模型参数 | 否 |

**重要说明：** 您的 API 密钥仅存储在您的本地设备上，本扩展不会将密钥发送给我们或任何第三方服务。密钥仅用于您授权的 AI 服务的 API 调用。

### 2.3 自定义内容

您创建和保存的内容：

- **自定义规则：** 您编写的提示词优化规则
- **提示词：** 您收藏或创建的提示词及其分类
- **标签和分类：** 用于组织提示词的自定义标签

### 2.4 使用统计

为改善用户体验，我们存储：

- 每日使用次数
- 总计使用次数
- 各提示词的使用频率



## 3. 数据使用方式

### 3.1 存储的数据

- **本地保存配置：** 让您的设置在不同会话中保持一致
- **提供核心功能：** 使用您的 API 配置调用相应 AI 服务完成提示词优化
- **个性化体验：** 根据您的偏好调整界面和功能

### 3.2 传输的数据

当您使用提示词优化功能时：

- 您的**原始提示词**会通过 API 请求发送给您配置的 AI 服务提供商
- 优化后的提示词由该服务返回，并显示在您的界面上

**注意：** 我们不拦截、记录或分析这些被加密的 API 通信内容。



## 4. 第三方服务

本扩展支持连接以下第三方 AI 服务：

- OpenAI (`api.openai.com`)
- Anthropic Claude (`api.anthropic.com`)
- Google Gemini (`generativelanguage.googleapis.com`)
- xAI Grok (`api.x.ai`)
- OpenRouter (`openrouter.ai`)
- Ollama（本地服务，默认 `localhost:11434`）
- 其他自定义 OpenAI 兼容端点

**重要说明：**

- **您自行配置：** 是否使用以及使用哪个 AI 服务完全由您选择
- **您的密钥：** 您需要提供自己的 API 密钥才能使用这些服务
- **直接通信：** 数据直接从您的浏览器发送到相应的 AI 服务，不经过我们的服务器
- **服务条款：** 使用这些服务时，您的数据受该服务的隐私权政策约束



## 5. 数据共享与出售

- **我们不向任何第三方出售您的数据。**
- **我们不向第三方共享您的数据。**
- **我们不使用您的数据进行广告定向或营销。**

唯一的数据传输是将您的提示词发送到您**自行选择并配置**的 AI 服务，以完成提示词优化功能。传输内容受该 AI 服务的隐私权政策约束。



## 6. 数据安全

我们采取以下措施保护您的数据：

- **本地存储：** 所有敏感数据（如 API 密钥）仅存储在您的本地设备上
- **无服务器收集：** 我们不收集任何用户数据到我们的服务器
- **加密传输：** API 通信通过 HTTPS 进行加密
- **本地验证：** 自定义规则通过客户端验证以防止潜在安全风险

您对数据安全采取的措施：

- **保护设备：** 请确保您的设备设置了锁定密码
- **安全浏览器：** 保持 Chrome 浏览器更新到最新版本
- **妥善保管密钥：** 不要与他人分享您的 API 密钥



## 7. 数据访问与控制

### 7.1 访问您的数据

您可以通过以下方式访问和管理您的数据：

1. 打开 Chrome 浏览器的扩展程序管理页面（`chrome://extensions`）
2. 找到 Chat Copilot 扩展
3. 点击"详细信息"
4. 在"扩展程序选项"中查看和修改您的设置

### 7.2 删除您的数据

要完全删除本扩展存储的所有数据：

1. 在 Chrome 中移除 Chat Copilot 扩展
2. 或在移除扩展时选择"清除浏览数据"选项
3. 或在 `chrome://settings/clearBrowserData` 中选择清除"扩展程序数据"

**注意：** 移除扩展后，所有本地存储的数据将被永久删除。



## 8. 网站访问权限

本扩展请求 `<all_urls>` 权限的原因是：

- **内容脚本注入：** 需要在各种 AI 平台（如 ChatGPT、Claude 等）页面上注入优化按钮和功能
- **平台适配：** 不同 AI 平台有不同的域名和页面结构

**我们承诺：**
- 不访问或读取任何与功能无关的网页内容
- 不收集您的浏览历史、密码或敏感信息
- 所有注入的代码仅在您主动使用功能时触发



## 9. 儿童隐私

本扩展不针对 13 岁以下儿童。我们不会故意收集儿童的个人信息。如果我们发现无意中收集了儿童的信息，我们将立即删除。

## 附录：Chrome 存储说明

本扩展使用的 `chrome.storage.local` API 具有以下特点：

- 存储容量上限约 5MB（取决于浏览器设置）
- 数据与您的 Chrome 用户配置文件绑定
- 数据在浏览器关闭后仍然保留
- 同步功能需要额外的 `storage.sync` 权限（本扩展未使用）
- 数据可被浏览器的"清除浏览数据"功能删除
