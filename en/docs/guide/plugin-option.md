## Settings Overview

### Optimization Engine

Choose the built-in engine or add custom models. You can configure the provider, API endpoint, and API key.

<p align="center">
  <img src="/ui-images/option/优化引擎.png" alt="Optimization Engine" width="80%" />
</p>

### Optimization Styles

Select prompt optimization styles to influence structure and tone.

<p align="center">
  <img src="/ui-images/option/优化风格.png" alt="Optimization Styles" width="80%" />
</p>

### Custom Optimization Rules

Add custom rules that are automatically appended to each prompt.

<p align="center">
  <img src="/ui-images/option/自定义优化规则%20.png" alt="Custom Optimization Rules" width="80%" />
</p>

### Floating Button

Configure display, click action, and hover drawer items for the floating button.

- Click action options: No action / Plugin Settings / [Prompt Plaza](/en/docs/guide/plugin-square.html) / [My Favorites](/en/docs/guide/plugin-favorite.html) / Prompt Optimization
- Hover drawer items: [Prompt Plaza](/en/docs/guide/plugin-square.html) / [My Favorites](/en/docs/guide/plugin-favorite.html) / Plugin Settings / Official Website

See details in [Floating Button](/en/docs/guide/floating-button.html).

<p align="center">
  <img src="/ui-images/option/悬浮按钮.png" alt="Floating Button" width="80%" />
</p>

### Prompt Plaza

Configure the sidebar display mode: push content or overlay. See [Prompt Plaza](/en/docs/guide/plugin-square.html) for details.

### Quick Access

Choose the sites shown in the [Plugin Panel](/en/docs/guide/plugin-popup.html). Supports select all, invert, and reset.
