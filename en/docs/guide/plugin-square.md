## Overview

Prompt Plaza curates high-quality prompts. It supports search and category filters so you can quickly find prompts to use.

![Prompt Plaza](/ui-images/ui-提示词广场.png)

## Entry

1. Open a supported platform page
2. Open the sidebar via the floating button or plugin panel
3. Select the "Prompt Plaza" tab

## Common Actions

- Search prompts: use the search box at the top
- Filter by category: Programming / Text / Image / PPT / Video / General
- Insert quickly: click "Insert" to fill the input box
- Copy content: click "Copy" to copy the prompt
- Add to favorites: click the star to save to [My Favorites](/en/docs/guide/plugin-favorite.html)
