## Overview

My Favorites helps you manage saved prompts. It supports search, filters, editing, and one-click insert.

![My Favorites](/ui-images/ui-我的收藏.png)

## Entry

1. Open a supported platform page
2. Open the sidebar via the floating button or plugin panel
3. Select the "My Favorites" tab

## Common Actions

- Search prompts: use the search box to find items
- Filter by category: Programming / Text / Image / PPT / Video / General
- Manage favorites: click the star to remove from favorites
- One-click insert: click "Insert" to fill the input box
- Copy content: click "Copy" to copy the prompt
- Edit favorites: click "Edit" to open the editor
- Export favorites: click "Export My Favorites" at the bottom

## Edit Favorites

Edit title, category, tags, and prompt content in the modal.
