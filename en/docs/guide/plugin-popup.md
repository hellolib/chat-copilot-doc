## Panel Overview

The plugin panel is the main control center of Chat Copilot. It provides engine selection, quick access, and usage stats.

<p align="center">
  <img src="/ui-images/ui-插件面板.png" alt="Plugin Panel" width="50%" />
</p>

## Key Features

- Engine selection: quickly switch prompt optimization engines
- Optimization stats: view today and total optimization counts
- Quick access: open commonly used AI platforms
- Feature entry: open Prompt Plaza, My Favorites, Settings, and more

## How to Open

1. Click the Chat Copilot icon in the browser toolbar
2. Use the panel to access features
