## Project Overview

Chat Copilot is an AI conversation enhancement assistant built on Chrome Extension Manifest V3. It helps you get higher-quality AI responses through intelligent prompt optimization.

## Feature Overview

- [Floating Button](/en/docs/guide/floating-button.html): open key features on any chat page
- [Plugin Panel](/en/docs/guide/plugin-popup.html): engine selection, stats, and quick access
- [Plugin Settings](/en/docs/guide/plugin-option.html): engines, styles, and personalization
- [Prompt Plaza](/en/docs/guide/plugin-square.html): curated prompts with search and filters
- [My Favorites](/en/docs/guide/plugin-favorite.html): save, edit, export, and insert prompts

## Features

### Intelligent Prompt Optimization

- **One-click Optimization**: optimize user prompts with a rule-based engine
- **Multi-dimensional Enhancement**: improve clarity, structure, and context completeness
- **Real-time Preview**: preview optimized prompts before sending
- **Custom Models**: use custom prompt optimization models
- **Multi-scenario Switching**: switch prompt styles for different scenarios

### Privacy Protection

- **Local Storage**: all data is stored locally
- **Zero Upload**: no conversation content is uploaded
- **Secure and Reliable**: your data stays under your control

## Tech Stack

- Chrome Extension Manifest V3
- TypeScript 5.3
- Webpack
