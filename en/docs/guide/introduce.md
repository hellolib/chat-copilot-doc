## Project Overview

Chat Copilot is an AI conversation enhancement assistant developed based on Chrome Extension Manifest V3 standards. It aims to improve user efficiency and experience when using AI platforms. Through intelligent prompt optimization, it helps users get more precise and higher-quality AI responses.

## Features

### Intelligent Prompt Optimization

- **One-click Optimization**: Intelligently analyze and optimize user input prompts based on a rule engine
- **Multi-dimensional Enhancement**: Provide optimization suggestions for clarity, structure, context completeness, and more
- **Real-time Preview**: Preview optimized prompts before sending, with one-click application
- **Custom Models**: Support custom prompt optimization models to meet personalized needs
- **Multi-scenario Switching**: Customize prompt styles for different scenarios

### Privacy Protection

- **Local Storage**: All data is stored locally
- **Zero Upload**: No conversation content is uploaded
- **Secure and Reliable**: Your private data is completely under your control

## Tech Stack

- Chrome Extension Manifest V3
- TypeScript 5.3
- Webpack
